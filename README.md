# Niespodzianka — Filip & Justyna

Wedding gift QR-code experience.

Scan the QR → watch a coconut fall → bunny presses boombox → Bad Bunny starts playing → at 03:20 the surprise reveals: **Filip & Justyna got tickets to Bad Bunny · 14.07.2026 · Stadion Narodowy, Warszawa**.

Open `index.html` directly or host as static site (Vercel / Netlify / GitHub Pages).
